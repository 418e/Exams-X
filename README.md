# Exams-X

Free and open-source tool to help students pass the National Exams of Georgia

## Run Locally

Clone the project

```bash
  git clone https://github.com/418e/Exams-X.git
```

Go to the project directory

```bash
  cd Exams-X/exams-x.com
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
